Intrinsic Resonance Holography (IRH) v21.1
==========================================

Welcome to the IRH Windows Installer!

This installer will set up IRH on your Windows system, including:

  - The complete IRH computational framework
  - Documentation and theory manuscripts
  - Jupyter notebooks for interactive exploration
  - Desktop application components

REQUIREMENTS
------------

Before proceeding, please ensure you have:

  1. Python 3.10 or later installed
     Download from: https://www.python.org/downloads/
     IMPORTANT: Check "Add Python to PATH" during installation

  2. Git (optional, for updates)
     Download from: https://git-scm.com/download/win

POST-INSTALLATION
-----------------

After installation, open a Command Prompt and run:

  1. irh test        - Verify the installation
  2. irh notebook    - Launch Jupyter notebooks
  3. irh shell       - Python shell with IRH

DOCUMENTATION

  - README.md - Quick start guide
  - IRH v21.1 Manuscript (Part 1, Part 2) - Full theory manuscript
  - notebooks/ - Interactive Jupyter notebooks
